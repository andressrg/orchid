The one feature = 80% of the value

Conversation-aware code review for agents

Every commit knows its session(s). A reviewing agent pulls the sessions that built a PR, reads why each change was made, asks Orchid fast Claude-powered follow-ups, and posts a review grounded in intent — not just the diff. The pieces exist in fragments; the loop does not yet.
